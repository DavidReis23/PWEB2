3- campoEntrada.classList.add("input-erro");
O classList é uma propriedade dos elementos do DOM que permite manipular as classes CSS aplicadas a um elemento.
Com ela, podemos adicionar, remover, alternar ou verificar classes sem precisar reescrever o atributo class inteiro.

4- O método document.createElement() cria um novo elemento HTML no documento.
Ele precisa receber como argumento o nome da tag a ser criada (por exemplo "li", "span", "button").

5- b — innerText retorna ou define apenas o texto visível ao usuário, enquanto innerHTML retorna ou define o conteúdo HTML do elemento, incluindo tags.
